# E-commerce Sales Analysis

##  Project Overview
This project is an end-to-end e-commerce sales analysis built to simulate a real-world data analyst workflow.

The goal of the project is to analyze transaction data, extract business insights, and present the results through a professional Power BI dashboard.

The project covers the full analytics lifecycle: data exploration, data cleaning, SQL analysis, and business-focused visualization.

-----

##  Business Questions
- Which countries generate the highest total revenue?
- Which product categories perform best?
- What are the peak and lowest sales months?
- Which payment methods are most commonly used?
- What is the average purchase value?

-----

##  Tools & Technologies
- **Excel** – initial data exploration
- **Python (Pandas)** – exploratory data analysis (EDA)
- **PostgreSQL** – data storage and SQL-based analysis
- **Power BI** – interactive dashboard and data visualization

-----

##  Key Insights
- **Sports** and **Toys** are the highest revenue-generating product categories
- **USA** and **France** are among the top-performing countries
- Sales peak in **January** and **July**, while **February** has the lowest sales volume
- The most frequently used payment methods are **UPI** and **Cash on Delivery**
- The average purchase value is relatively high, indicating a premium-oriented customer base

-----

##  Dashboard
The Power BI dashboard includes:
- KPI cards for Total Revenue, Average Purchase Value, and Total Transactions
- Monthly sales trend analysis
- Top 5 countries by revenue
- Revenue distribution by product category
- Payment method performance analysis

The dashboard is designed for executive-level decision-making and quick insight discovery.
